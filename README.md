# Image Recognition App 
Developed an Android app for real-time image recognition using Java and ML Kit. The app can label objects in images captured via the device camera, providing instant feedback and visualization.
# Key Features :
- Real-time image labeling using ML Kit.
- Captures images directly from the camera.
- Displays recognized object labels with confidence scores.
- Built with Android Studio and Java for smooth performance.

# Tech Stack:
- Language: Java
- Framework: Android SDK
- ML Tool: ML Kit (Image Labeling)
